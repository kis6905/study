### 1. 엘레베이터에 필요한 속성과 기능을 만들어보세요.
- 속성
  - 현재층수
  - 목적지층수
  - 문 닫힘 여부
- 기능
  - 문 닫힘
    - 문이 열려있을때만 "닫혔습니다" 출력
  - 문 열림
    - 문이 닫혀있을때만 "열렸습니다" 출력
  - 이동 기능
    - 현재 층수에서 목적지 층수에 따라 "올라갑니다" / "내려갑니다" 출력
    - 문이 열려있으면 "문이 열려있습니다" 출력
```

}
```
